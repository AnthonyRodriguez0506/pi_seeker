# pi_seeker

![Portada del proyecto](/images/pi.webp)

## Descripción
**pi_seeker** es un algoritmo que escanea los dígitos de π (pi), los convierte a binario y luego los mapea a caracteres alfabéticos para buscar patrones ocultos.  
Este proyecto explora la teoría de números y la codificación de datos encontrando posibles mensajes o secuencias dentro de los dígitos de pi.

## Uso

### 1. Clonar el proyecto
```bash
git clone https://github.com/AnthonyRodriguez0506/pi_seeker.git
```  

### 2. Ejecutar el Script
```bash
python3 test.py
```  

El archivo pi_digits.txt contiene los dígitos de π que el algoritmo utilizará para analizar patrones.

### Importante

Asegúrate de que el archivo pi_digits.txt esté en la misma carpeta que el script test.py.
Puedes reemplazar este archivo con más dígitos de π si quieres ampliar el análisis.
